# Python_MiniProject
This repo consist of Python based miniproject

## Automated Birthday Wisher
A Python-based automation tool that monitors a database of birthdays and sends personalized email greetings on the correct dates. This project eliminates manual tracking by integrating file handling with the SMTP email protocol to ensure timely wishes.

## Kanye Says - API Quote Generator
A desktop GUI application built with Python that connects to an external API to fetch and display random quotes from Kanye West. This project demonstrates how to bridge backend data fetching with a frontend user interface.
